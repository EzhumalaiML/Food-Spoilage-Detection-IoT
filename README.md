# Food Spoilage Detection System (IoT)

This project detects food spoilage using an MQ-6 gas sensor and displays the status using LEDs, a buzzer, and a web interface over WiFi.

## Features
- MQ-6 gas sensor monitoring
- Red & Green LED indication
- Buzzer alert for spoiled food
- Live web page display using WiFi
- Arduino + WiFiNINA

## Hardware Used
- Arduino (WiFi supported)
- MQ-6 Gas Sensor
- Red LED, Green LED
- Buzzer
- Resistors & Jumper wires

## Working
- If gas level exceeds threshold → Food is NOT good
- Else → Food is good
- Status visible on LEDs and web browser

## Applications
- Food storage monitoring
- Smart kitchens
- Food safety systems
